Module 11 – Data Visualization with Matplotlib
This repository contains my work for Module 11 of the Python for Data Analysis course.
Assignment Overview
This assignment builds a comprehensive set of data visualizations for a fictional retail company called SunCoast Retail. Using synthetically generated quarterly sales data from 2022–2023, the script produces 13 distinct charts and a written business insights summary.
Files Included

module11-assignment.py – Main Python script with all completed tasks

What I Practiced

Creating bar charts, line charts, scatter plots, pie charts, histograms, and box plots with Matplotlib
Customizing chart aesthetics including colors, labels, titles, legends, and gridlines
Using numpy to generate and manipulate sample data for visualization
Plotting multi-series data and stacked charts to compare categories over time
Annotating charts with key data points to highlight trends
Organizing multiple plots using plt.subplot() layouts
Writing a structured business insights summary based on visual analysis
